## This Project — Drifting Pixels Landing Site

- **Framework:** Next.js
- **Purpose:** Landing page / portal for the Drifting Pixels brand
- **Theme:** Inspired by Bungie's Marathon aesthetic — "Graphic Realism" meets cyberpunk
- **Key visual concept:** "Drifting pixels" — pixels that drift, float, and animate across the page
- **Vibe:** Very cool, animation-heavy, visually striking
- **Style reference:** Bungie's Marathon (bungie.net/marathon)
  - Y2K cyberpunk meets acid graphic design
  - Vibrant, bold, alien colour palette — not cartoony but aggressively clear
  - Glitchy UI elements, glitch content/transitions
  - Strong graphic design and colour statements
  - View transitions, slide animations, opacity fades, transform-based motion
  - Mesh-based grid layouts, variable font systems
  - "Wipeout meets Ghost in the Shell" energy
  - Influences: Alberto Mielgo, Spider-Verse aesthetic, product/apparel design
- **Additional direction:** Retro 8-bit pixel art elements woven into the Marathon-style aesthetic — a fusion of pixel nostalgia with modern cyberpunk graphic realism
